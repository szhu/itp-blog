Faking the News

Unofficial Course Info • Prof. Benjamin Moskowitz

This class is about why fake news exists, how it is created, and how it spreads. Discussion is sometimes technical (e.g., we learn about software used to make deepfakes), historical (e.g., what are the origins of news media), and social (e.g., how does social media change people’s perception of the truth). Homework is technical — we learn how to make our own deepfakes!

Physical Computing, Introduction

Course Info • Prof. Daniel Rozin

This class is an introduction to understanding and making simple electronics circuits and IoT devices. We learn about how to calculate voltage, what resistors do, to wire things up on a breadboard, how to solder, how to debug circuits, and how to write code for an Arduino and have it interact with circuits, sensors, and computers.

Population Infinite: The Future of Identity

Course Info • Profs. Dan Oved & Anastasis Germanidis

Class about how technology is changing our concept of identity. We learn about how identity can be simulated, hidden, abused, redefined, repurposed, etc. Homework is technical — we learn how to use various tools that we can use to play around with identity. These tools are useful outside of this class too — they include the test automation library Puppeteer and the text ML model GPT-2.
