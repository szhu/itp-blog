Physical Computing

This class is an introduction to understanding and making simple electronics circuits and IoT devices. We learn about how to calculate voltage, what resistors do, to wire things up on a breadboard, how to solder, how to debug circuits, and how to write code for an Arduino and have it interact with circuits, sensors, and computers.
